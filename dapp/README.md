## Superfluid Flow Splitter dApp

### Prerequisite
- `npm` or `pnpm`

### How to use

First install dependencies: `pnpm install`.

To run the development server: `pnpm dev`. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

To build: `pnpm build`.
